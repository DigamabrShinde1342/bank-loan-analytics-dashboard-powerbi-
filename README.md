# 🏦 Bank Loan Analytics Dashboard | Power BI

## 📌 Project Overview

The Bank Loan Analytics Dashboard is an end-to-end Power BI project designed to analyze loan portfolio performance, borrower behavior, repayment trends, and credit risk.

The objective of this project is to help business stakeholders monitor lending operations, evaluate portfolio health, identify risky loans, and make data-driven lending decisions.

---

## 🎯 Business Problem

Banks process thousands of loan applications every year. Without proper analysis, it becomes difficult to:

* Monitor loan portfolio performance
* Track loan repayments
* Identify high-risk borrowers
* Measure loan profitability
* Analyze lending trends
* Support strategic decision-making

This dashboard provides a centralized view of key lending KPIs and portfolio performance.

---

## 📊 Dashboard Pages

### 1️⃣ Summary Dashboard

Provides a high-level overview of:

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income (DTI) Ratio
* Good Loan vs Bad Loan Analysis
* Loan Status Performance
* Repayment Monitoring

### 2️⃣ Overview Dashboard

Provides deeper business insights through:

* Monthly Loan Application Trends
* Regional Loan Distribution
* Loan Term Analysis
* Home Ownership Analysis
* Loan Purpose Breakdown

## 🔍 Key Business Insights

### ✔ Strong Portfolio Performance

The bank funded $435.8M and received $473.1M, indicating positive portfolio returns.

### ✔ Healthy Loan Portfolio

86.18% of loans are classified as Good Loans, reflecting strong repayment performance.

### ✔ Risk Concentration

Only 13.82% of loans are categorized as Bad Loans, but they contribute significantly to portfolio losses.

### ✔ Growing Customer Demand

Loan applications increased steadily throughout the year, indicating strong market demand.

### ✔ Debt Consolidation Dominates

Debt Consolidation represents the largest loan purpose category, highlighting customer demand for debt restructuring.

### ✔ Higher Risk Indicators

Current loans exhibit the highest average interest rates and DTI ratios, making them important candidates for risk monitoring.

## 🚀 Business Recommendations

### Improve Risk Assessment

Implement stricter screening for borrowers with high DTI ratios and higher risk grades.

### Monitor Current Loans

Track current loans proactively to reduce future charge-offs.

### Focus on High-Performing Segments

Expand lending to customer segments with strong repayment histories.

Develop predictive risk models for early default detection.
